# Datei: .gitignore

## Zweck
Schließt generierte und lokale Dateien vom Git-Tracking aus.

## Inhalt 1:1
- `/target`: Rust Build-Ausgaben
- `**/*.rs.bk`: Backup-Dateien
- `Cargo.lock`: Lockfile (in diesem Projekt ausgenommen vom Tracking)
- `.DS_Store`: macOS Finder-Dateien
- `.idea`: JetBrains IDE-Dateien
- `.vscode`: VS Code Workspace-Dateien
